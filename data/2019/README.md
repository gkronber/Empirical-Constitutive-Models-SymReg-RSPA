Folder `orig` contains original files from Kabliman E, Kolody AH, Kommenda M, Kronberger G.: _Prediction of stress-strain curves for aluminium alloys using symbolic regression._ AIP Conference Proceedings 2113, 180009, 2019 (https://doi.org/10.1063/1.5112747).

The files `AA6082_all_temps_combined.csv` and `AA6082_all_temps_combined_tomax.csv` contain all data from the individual files and cut off at the maximum stress value. The two files are used for plotting mainly.

The two files contain stress-strain measurements for phip in (0.001, 0.01, 0.1, 1.0, 10) and temperture in (350, 375, 400, 425, 475, 500). 

Data are shared with kind permission of LKR Light Metals Technologies, AIT Austrian Institute of Technology GmbH.
